## How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/Data-Analysis-Projects.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
